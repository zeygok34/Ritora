# Ritora (EduVise Learn+) 📱🤖

**Ritora**, özel gereksinimli çocukların öğrenme süreçlerini desteklemek amacıyla geliştirilmiş, yapay zeka destekli ve oyunlaştırma tabanlı bir mobil eğitim platformudur.

## 🌟 Öne Çıkan Özellikler
- **AI Öğrenme Asistanı:** Groq API entegrasyonu ile çocukların seviyesine uygun, kişiselleştirilmiş geri bildirimler sağlayan akıllı ajanlar.
- **Oyunlaştırma:** Öğrenme motivasyonunu artırmak için tasarlanmış ödül sistemleri ve interaktif görevler.
- **İlerleme Takibi:** Eğitimciler ve veliler için detaylı analiz panelleri.

## 🛠️ Teknik Mimari
- **Frontend:** Flutter & Dart (Modern UI/UX prensipleriyle geliştirildi).
- **Backend:** Firebase (Veritabanı yönetimi ve kullanıcı yetkilendirme).
- **AI Engine:** Groq API (Doğal Dil İşleme ve akıllı içerik üretimi).

## 🚀 Kurulum
Projenin yerel ortamınızda çalışması için Flutter SDK'nın yüklü olması gerekmektedir:

1. Depoyu klonlayın: `git clone https://github.com/zeygok34/Ritora.git`
2. Paketleri yükleyin: `flutter pub get`
3. Uygulamayı çalıştırın: `flutter run`
